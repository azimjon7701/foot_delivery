1. Admin – barcha huquqlarga ega
2. Ofitsiant
- Taomlarni qo’shish, o’zgartirish, o’chirish
- Buyurtmalarni ko’rish, qabul qilish va yuborish
3. User
- Taomnomani ko’rish
- Buyurtma berish
